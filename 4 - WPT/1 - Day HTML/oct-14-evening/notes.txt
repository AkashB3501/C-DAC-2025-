HTML: Hyper Text Markup Language

HTML is for creating skeleton of a web page
Using HTML we can display things on a web page

Scripting lang: no compiler JS, PHP
Programming lang: compile C, C++, Java

Markup lang: We can't write logics with markup langs
             1. store the data -> XML -> to store the configs 
             2. display the data -> HTML

HTML syntax: we have to make use of tags only

<tag-name> opening tag

</tag-name> closing tag

types of tags:
1. empty tag -> opening only 
2. container tag -> opening and closing both 

HTML provides the predefined tags only 
HTML is not case sensititve
extension will be .html .htm
html code will be read by browser and by reading the html code browser will display the data accordingly
HTML is error free lang


<html>
	<head>
		<title>My first page</title>
	</head>
	<body>
		<p>Hello world</p>
	</body>
</html>

bydefault everything will be auto aligned at the left side 

to customize a tag we are getting several properties/attributes 
eg: <p align="center">Hello world</p>

h1, h2, h3, h4, h5, h6
